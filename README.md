Installation
========
`cd frontend; npm install;`
`cd ../backend; npm install`

Running the frontend
========
`cd frontend`

... then any of the following

  `npm start`
    Starts the development server.

  `npm run build`
    Bundles the app into static files for production.

  `npm test`
    Starts the test runner.

Running the backend
===================
`cd backend`

then
`make run-local`

Testing the backend
`make test-local`
